Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tags Name Purpose
h1–h6 headings include a heading (level 1–6)
p paragraph include a paragraph of text
table table include a table
th table header make a table header
tr table row create a table row
td table data include a table data cell
header header create a header container
div division create a container for block elements
ol ordoned list classify list's elements by importance with numbers
ul unordoned list create a list of unordoned items peceded by dots
span span container for inline elements
blockquote blockquote a space where we write quotations
<!--...--> comment write a comment that appears only in the text editor

Inline Elements

Tags Name Purpose Example Result
em emphasized make emphasized text <em>Technical Sophistication</em> Technical Sophistication
strong bold make bold text <bold1>Technical Sophistication</bold> Technical Sophistication
a anchor make hyperlink <a href"...">Technical Sophistication</a> Technical Sophistication
img image include a image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code code make text letters same width <code>Technical Shophistication</code> Technical sophistication
span span create a container for inline elements <span></span> Can't really dispaly a result of span code
li list element create a list element <li>Thecnical Sophistication</li>
